# proyecto-git

Proyecto de práctica de control de versiones con Git y GitHub.

## Descripción

Este repositorio demuestra el uso de Git: commits, ramas, merges, resolución de conflictos y sincronización con un repositorio remoto.

## Estructura

```
proyecto-git/
├── README.md
├── index.html
├── css/
│   └── estilos.css
├── js/
│   └── app.js
└── docs/
    └── COMANDOS-GIT.md
```

## Historial de ramas

El proyecto incluye las siguientes ramas:

- `main` - Rama principal
- `develop` - Rama de desarrollo
- `feature/estilos` - Estilos CSS
- `feature/saludo` - Funcionalidad interactiva
- `feature/contacto-v2` - Sección de contacto (con conflicto resuelto)
- `feature/documentacion` - Documentación ampliada
- `hotfix/typo-readme` - Corrección en README
